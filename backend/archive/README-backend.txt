Backend (Spring Boot + Gradle - groovy)
-------------------------------------
- Java 17 required.
- To generate Gradle wrapper (if you don't have it), run:
    gradle wrapper

- To run:
    ./gradlew bootRun
  or
    gradle bootRun

- H2 console available at http://localhost:8081/h2-console
  JDBC URL: jdbc:h2:mem:demo
